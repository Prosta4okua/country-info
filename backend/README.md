## Installation

1. Clone the repository:

```sh
git clone https://github.com/your-username/country-info.git
cd cinema-scout
```
2. Install dependencies:
```sh
yarn install
```

## Running the Application

2. Start the application in development mode:
```sh
yarn start:dev
```
